# 二つ名メーカーDX

GitHub Pagesで公開できる、無料の二つ名生成ツールです。

## 内容

- `index.html` トップページ
- `style.css` デザイン
- `script.js` 生成ロジック
- `words.js` カテゴリ別単語データ

## 主な機能

- 1,350,385通り以上の組み合わせ
- 厨二・和風・かわいい・最強・ネタカテゴリ
- 一度に3つ生成
- レア演出
- コピー機能
- Xシェア機能
- 生成履歴
- スマホ対応

## 公開方法

1. ZIPを解凍します。
2. 中のファイル一式をGitHubリポジトリにアップロードします。
3. GitHub Pagesを有効化します。

## AdSenseについて

`index.html` 内の「広告スペース」と書かれている場所を、AdSense審査通過後に広告コードへ差し替えてください。
